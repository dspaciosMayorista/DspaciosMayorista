# Lote 2 — ejemplos de programas de terceros

Sube aquí los nuevos programas de proveedor para analizar si la estructura
actual los cubre o si hace falta algo distinto (p. ej. matriz fecha×precio).

- `01-original-proveedor/` → el archivo **tal como lo envía el proveedor**.
- `02-publicado-dspacios/` → la versión **publicada/transformada** (si la tienes),
  para comparar neto vs. precio de venta.

Formatos: **.docx** o **.pdf** (se leen bien). El **.doc** viejo (binario) no se
puede extraer — conviértelo a .docx o .pdf antes de subirlo.
