# Programas (circuitos de proveedor) — material de referencia

Carpeta para los **PDF de ejemplo** de programas completos que arma un
proveedor (tipo circuito: itinerario día por día, varias ciudades, hoteles por
tramo, qué incluye/no incluye, salidas, precios).

## Sube aquí el/los PDF de ejemplo

```
dspacios-travel/docs/programas/ejemplo-1.pdf
dspacios-travel/docs/programas/ejemplo-2.pdf   (si tienes más de un estilo)
```

Cuando estén arriba, avísame: los leo, identifico **qué información necesitamos
extraer** y te propongo cómo modelarlo (tablas + módulo de montaje), igual que
hicimos con Bloqueos / Porción / Servicios.

> Entre más ejemplos (de distintos proveedores/estilos), mejor para que el
> modelo cubra los casos reales y no quede amarrado a un solo formato.
